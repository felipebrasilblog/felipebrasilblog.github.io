A.I. Free

01/06/2026 In English

This blog posts content is 100% A.I. free.

And what does that mean? What is the point of this?

Once I wrote a text and asked for a grammar check in my text on Gemini. It just changed the word order, created synonyms, correct expressions and created new punctuation creating a different meaning on my whole text. And I only asked a grammar check.

I didn`t use the I.A. edited text because it was really not natural. It was not my text, but a Frankenstein of decisions made by a machine. Yeah, maybe the original repetition of terms in my text was annoying, maybe I should separate some sentences… maybe using the reticence was utterly wrong. But IT WAS MY TEXT.

My text, wrote by a human, with mistakes, some mistakes here and there. My way to write. I am not a professional academic master of English/Portuguese language that perform in a perfect way and deliver the most correct text according to language rules.

I am human, so far, I make mistakes.

While the “perfect” text wrote by the I.A., it was clearly not me. I don’t use those words, I don’t conjugate verbs that way, the meaning of some ideas changed. That was not me.

Today I use I.A. in this blog for 2 things: Adding paragraph tag using HTML and to use the list tag to order lists. I write the text, ask Gemini to add those HTML tags without manipulate the content. It is a job that would take me 5 to 10 minutes to manually add these reference tags in the beginning and end of each paragraph, A.I. add them in 5 secs.

After that I manually edit the post, generate the files and organize the website in a logical construction created by me. It takes between 5 and 10 minutes to create the files, copy patterns, organize documents, save on github and upload the server to the web.

I also use I.A. to turn charts and tables with lots of content using a script created for that table. This really speed up my work, for example, I have a post about my top 100 musical artists. Editing it myself would take at least one hour. With A.I. I wrote the script and my tablet was formatted in less than 10 seconds.

For this kind of edit, repetitive adding tags, configurating tables and lists, I am 100% A.I.

Now, creative content, my ideas, my texts, my pictures, my videos, etc. This is all handcrafted. It takes time and mor effort to write from scratch, to think about a topic, to elaborate the ideas. But it is completely worth it.
